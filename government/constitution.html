Constitution Bylaws Judicial Code Election Code Bills & Resolutions

The Hanszen College Constitution outlines rules and goals for the governance of our college. The Hanszen Constitution was last updated in Febuary 2017. You can view it below or download it here.

The Hanszen College Bylaws outline procedural rules for running the college, room draw, parking draw, and more. The Hanszen Bylaws was last updated in February 2018. You can view it below or download it here.

The Hanszen College Judicial Code outlines basic guidelines for how the College Court operates. The Hanszen Judicial Code was last updated November 2016. You can view it below or download it here.

The Hanszen College Election code outlines how we elect students for positions in Hanszen goverment. The Election Code was last updated in November 2016. You can view it below or download it here.

Bills and Resolutions are a way for the college to implent procedures for a time span of one (1) academic year.

The last bill was added in February 2019. A passed bill is considered in effect until the end of the first cabinet of the following academic year. You can view a list of current bills below or download it here.

There are currently no resolutions in effect. A resolution is considered in effect until the end of the first cabinet of the following academic year.
